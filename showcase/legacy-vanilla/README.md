# Legacy Vanilla Showcase Archive

This folder preserves the pre-Angular showcase implementation that originally lived at `showcase/*.html`, `showcase/css/`, and `showcase/js/`.

## Contents

- HTML routes: `index.html`, `about.html`, `dashboard.html`, `privacy.html`, `support.html`
- Legacy styles: `css/`
- Legacy scripts: `js/`

## Asset Paths

Legacy pages now reference `../assets/...` so shared logos/icons/providers continue to resolve from `showcase/assets/` after archival.
