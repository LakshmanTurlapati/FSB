<objective>
Implement BG-04 user-initiated navigation pause emission on agent-owned tabs by extending the existing `chrome.webNavigation.onCommitted` listener at `background.js:2464`. Detect transitionType in {typed, auto_bookmark, reload, link} and emit a LOG-04 `agent-tab-user-navigation` event when the affected tab is owned by a non-legacy agent. Suppress the emission when the registry indicates an agent-initiated navigation occurred on the same tab within 500ms (Phase 242 `back` produces transitionType auto_bookmark — false-positive). Add `stampAgentNavigation(tabId)` helper to agent-registry plus stamps at every programmatic chrome.tabs.update / chrome.tabs.goBack invocation in the extension.

This is an EMISSION-only signal (CONTEXT specifics line 67-68). No actuation; no session.status mutation; no pause/resume primitive. The signal lives in the LOG-04 ring buffer for downstream observation.

Purpose: Multi-agent observability — diagnostics records when a user grabs the wheel from an MCP agent so future tooling (dashboard, agent loop) can adopt the signal without re-scanning webNavigation events.
Output: Extended listener + registry helper + 3-case test green.
</objective>
