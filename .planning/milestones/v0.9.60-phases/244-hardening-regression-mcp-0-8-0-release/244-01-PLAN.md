Registry public API (from extension/utils/agent-registry.js):
- registerAgent({ connectionId? })           -> { ok:true, agentId } | { ok:false, error:'AGENT_CAP_REACHED', cap, active }
- bindTab(agentId, tabId, { forced? })       -> { ok:true, ownershipToken } | { ok:false, error }
- isOwnedBy(tabId, agentId, ownershipToken?) -> boolean
- releaseTab(tabId)                          -> idempotent; only fully releases the agent when its pool.size === 0
- stagedRelease(connectionId)                -> stages a delayed release keyed by connection_id
- hydrate()                                  -> rebuilds in-memory state from chrome.storage.session 'fsbAgentRegistry' envelope, drops ghosts via chrome.tabs.query reconciliation
- setCap(n)                                  -> integer-clamped 1..64
- getCap()                                   -> current cap

Storage envelope shape (chrome.storage.session 'fsbAgentRegistry'):
{
  records: { [agentId]: { agentId, tabIds: number[], connectionId?, createdAt, ... } },
  tabOwners: { [tabId]: agentId },
  ownershipTokens: { [tabId]: token },
  stagedReleases: { [connectionId]: { agentIds: string[], expiresAt: number } }
}

Test harness surface (from tests/fixtures/run-task-harness.js):
- createStorageArea(initial)         -> async chrome.storage.session-shaped store with _dump()
- installChromeMock({ tabs, storage }) -> { chrome, restore }; tabs.query/get backed by `tabs` arg
- installVirtualClock()              -> { advance(ms), restore, now() } for RECONNECT_GRACE_MS time-travel
