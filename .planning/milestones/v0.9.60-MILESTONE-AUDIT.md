1. `fsb-mcp-server@0.8.0` is tag-ready, not published by this closeout run.

   Phase 244 intentionally stopped before `npm publish`; the publish action remains user-gated. This is recorded in `244-VERIFICATION.md` and requirements traceability as `Verified, publish user-gated`.

2. Live unowned-target `switch_tab` recovery could not be reproduced in the current browser profile.

   Browser-created normal tabs were immediately owned by `legacy:sidepanel`, so live `switch_tab` attempts correctly rejected as `Tab ownership`. The unowned-target claim path is covered by `tests/recovery-tools-bootstrap.test.js`, and live cross-owner protection behaved correctly.

   The original P2 checklist asks for five real tasks longer than five minutes. Automated lifecycle, heartbeat, persistence, and timeout coverage is green; the live soak is accepted as deferred unless the operator wants a separate soak session.
