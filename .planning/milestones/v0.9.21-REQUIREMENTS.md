## v0.9.21 Requirements

### Shared Surface Cohesion

- [x] **COH-01**: Shared buttons, badges, cards, headers, and status indicators follow one visual treatment across popup, sidepanel, control panel, and dashboard
- [x] **COH-02**: The retouch keeps FSB's existing tone and color direction rather than introducing a different visual language
- [x] **COH-03**: Audited surfaces no longer show obvious spacing, sizing, or typographic drift in their default states

### Sidepanel

- [x] **SID-01**: Sidepanel header, history view, composer, and footer present a cleaner, more consistent hierarchy with no clipped or awkward states
- [x] **SID-02**: High-frequency sidepanel chat interactions look polished without changing behavior

### Popup

- [x] **POP-01**: Popup controls, status display, and composer styling align with the sidepanel where patterns are shared
- [x] **POP-02**: Hover, active, disabled, and busy states on popup controls feel deliberate instead of visually inconsistent

### Control Panel & Dashboard

- [x] **CP-01**: Control panel navigation, headers, cards, forms, analytics, and utility sections use a consistent visual system
- [x] **CP-02**: Settings and dashboard areas no longer show broken spacing, mismatched surfaces, or low-quality responsive behavior
- [x] **DSH-01**: The dashboard keeps the same FSB aesthetic while feeling more polished and internally consistent

### Overlay Feedback

- [x] **OVR-01**: Link and inline text targets can be emphasized in a way that reads like text highlighting rather than an oversized field box
- [x] **OVR-02**: Larger controls such as inputs and buttons still receive clear, well-fitted target overlays
- [x] **OVR-03**: The highlight animation remains smooth and consistent in both the live page and dashboard DOM stream

### Final QA

- [x] **QA-01**: All touched UI surfaces remain fully usable after the retouch with no interaction or theme regressions
- [x] **QA-02**: Obvious visual inconsistencies and low-quality states identified during the milestone are resolved before ship
