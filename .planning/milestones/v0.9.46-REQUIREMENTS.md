# Requirements Archive: FSB v0.9.46 Site Discoverability (SEO + GEO)

**Defined:** 2026-04-30
**Shipped:** 2026-05-02
**Status:** SHIPPED — 24/24 requirements satisfied (22 automated + 2 manual UAT verified live post-deploy)

## Milestone Goal (Outcome)

Made `full-selfbrowsing.com` discoverable to traditional search engines and generative AI search by prerendering the four marketing routes and shipping LLM/crawler-aware root files. Pre-milestone the site returned only the literal string "FSB" to non-JS crawlers; post-milestone every route returns route-specific prerendered HTML with structured data and the site exposes `/robots.txt`, `/sitemap.xml`, `/llms.txt`, `/llms-full.txt`. Verified live: Google Rich Results Test detects 1 valid `SoftwareApplication` (0 errors); GPTBot-UA crawls of `/`, `/about`, `/privacy`, `/support` return route-specific titles + canonicals.
