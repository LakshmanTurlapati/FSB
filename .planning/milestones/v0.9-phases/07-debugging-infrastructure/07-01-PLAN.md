<task type="auto">
  <name>Task 4: Integrate actionRecorder.record() into tool handlers</name>
  <files>content.js</files>
  <action>
Integrate actionRecorder.record() calls into all tool handlers in the tools object to ensure every action is recorded.

For each tool handler (click, type, selectOption, toggleCheckbox, pressEnter, hover, focus, blur, doubleClick, rightClick, clearInput, etc.), add actionRecorder.record() calls:

1. At the START of each handler, capture startTime: const startTime = Date.now();

2. Before returning success/failure, call actionRecorder.record() with:
   - tool: the tool name (e.g., 'click', 'type')
   - params: the original params passed to the tool
   - selectorTried: the selector(s) attempted
   - selectorUsed: the selector that worked (or null if none)
   - elementFound: boolean indicating if element was found
   - elementDetails: captureElementDetails(element) result
   - coordinatesUsed: { x, y } if coordinates were used
   - coordinateSource: 'selector' | 'fallback' | 'direct' depending on how coordinates were obtained
   - success: boolean indicating if action succeeded
   - error: error message if failed (null if success)
   - hadEffect: verification result from verifyActionEffect if applicable
   - diagnostic: generateDiagnostic() result if failed
   - duration: Date.now() - startTime

Pattern for click tool (apply similar to others):
```javascript
click: async (params) => {
  const startTime = Date.now();
  const selectorTried = params.selector;
  let element = null;
  let selectorUsed = null;
  let coordinatesUsed = null;
  let coordinateSource = null;

  // ... existing element finding logic ...
  // Set element, selectorUsed when found
  // Set coordinatesUsed, coordinateSource when using coordinates

  // ... existing click execution ...

  // On success:
  actionRecorder.record(null, 'click', params, {
    selectorTried,
    selectorUsed,
    elementFound: !!element,
    elementDetails: captureElementDetails(element),
    coordinatesUsed,
    coordinateSource,
    success: true,
    hadEffect: effectResult?.hadEffect,
    duration: Date.now() - startTime
  });

  // On failure:
  actionRecorder.record(null, 'click', params, {
    selectorTried,
    selectorUsed: null,
    elementFound: false,
    elementDetails: null,
    coordinatesUsed,
    coordinateSource,
    success: false,
    error: errorMessage,
    diagnostic: generateDiagnostic('elementNotFound', { selector: selectorTried }),
    duration: Date.now() - startTime
  });
}
```

Integrate into these tool handlers:
- click (primary - most complex with coordinate fallback)
- type (with selector and text params)
- selectOption (dropdown selection)
- toggleCheckbox (checkbox toggle)
- pressEnter (form submission)
- hover, focus, blur (element interaction)
- doubleClick, rightClick (click variants)
- clearInput (input clearing)
- scroll (if uses element selector)

For tools that don't use selectors (navigate, searchGoogle, refresh, goBack, goForward), record with elementFound: true, selectorTried: null.

This ensures EVERY action execution is captured for debugging and replay.
