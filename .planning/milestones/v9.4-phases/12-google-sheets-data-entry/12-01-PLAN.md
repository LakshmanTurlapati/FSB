**1. `getAccumulatedJobData()` function:**
Read `fsbJobAccumulator` from `chrome.storage.local`. Flatten all jobs from completed companies. Return `{ jobs: [...], totalJobs: N, searchQuery: '...', companies: [...] }`. Filter only `status === 'completed'` entries with valid `.jobs` arrays. Each job has fields: title, company, location, datePosted, description, applyLink.

**2. `formatJobDataForPrompt(jobData)` function:**
Convert the job data array into a compact table string for AI prompt injection. Format:
```
Row 2: Title: "SWE Intern" | Company: "Microsoft" | Location: "Redmond, WA" | Date: "2026-02-15" | Desc: "Design and implement..." | ApplyLink: https://...
Row 3: ...
```
Cap description at 200 chars. Use "N/A" for any missing field. Return the full string. Also return the count of rows.

**3. `detectSheetsIntent(task)` function:**
Check if the user's original task implies writing to a Google Sheet. Return true if the task contains keywords like: "spreadsheet", "sheet", "sheets", "google sheet", "google sheets", "write to sheet", "put in sheet", "add to sheet", "create a sheet", "fill sheet", "populate sheet", "make a spreadsheet". Also return true if the task contains "and" followed by any of these keywords. This should be a simple keyword check, not complex NLP.

**4. `findExistingSheetsTab()` function:**
Use `chrome.tabs.query({})` to find tabs matching `docs.google.com/spreadsheets/d/` URL pattern (NOT the home page or template page). Return the first matching tab (tabId + url), or null if none found. This supports the "I have a Sheets tab open" flow.

**5. `startSheetsDataEntry(sessionId, session)` async function:**
This is the core orchestrator. Flow:
a. Call `getAccumulatedJobData()`. If jobs.length === 0, log warning and return (nothing to write).
b. Call `formatJobDataForPrompt()` to create the prompt-injectable data string.
c. Determine sheet target:
   - Check if user's original task mentions a specific Sheets URL (regex for `docs.google.com/spreadsheets/d/`). If yes, use that URL.
   - Else call `findExistingSheetsTab()`. If a tab is found, plan to switch to it.
   - Else, plan to create a new sheet via `https://docs.google.com/spreadsheets/create`.
d. **Parse user-customizable column selection (SHEETS-02):**
   Call `parseCustomColumns(session.multiSite?.originalTask || session.task)` to determine if the user requested specific columns. Default columns are `['Title', 'Company', 'Location', 'Date', 'Description', 'Apply Link']`. The parser scans the original task for patterns like:
   - "only title, company, and apply link" -> filter to those 3 columns
   - "columns: title, location, link" -> filter to those columns
   - "just the title and company" -> filter to those 2 columns
   - Keyword mapping: "title"/"role"/"position" -> Title, "company"/"firm"/"employer" -> Company, "location"/"city"/"place" -> Location, "date"/"posted"/"when" -> Date, "description"/"desc"/"details"/"summary" -> Description, "link"/"url"/"apply"/"apply link" -> Apply Link
   - If no custom column keywords found, return all 6 default columns.
   - Store result as `customColumns` array.

   **`parseCustomColumns(task)` function** (place near `detectSheetsIntent`):
   ```javascript
   function parseCustomColumns(task) {
     const defaults = ['Title', 'Company', 'Location', 'Date', 'Description', 'Apply Link'];
     const taskLower = task.toLowerCase();
     // Look for explicit column restriction patterns
     const restrictionPatterns = [
       /(?:only|just)\s+(?:the\s+)?(?:columns?\s+)?(.+?)(?:\s+(?:in|into|to|on)\s+|$)/i,
       /columns?:\s*(.+?)(?:\s+(?:in|into|to|on)\s+|$)/i,
       /(?:with|include)\s+only\s+(.+?)(?:\s+(?:in|into|to|on)\s+|$)/i
     ];
     const columnAliases = {
       'Title': ['title', 'role', 'position', 'job title', 'job name'],
       'Company': ['company', 'firm', 'employer', 'organization', 'org'],
       'Location': ['location', 'city', 'place', 'where'],
       'Date': ['date', 'posted', 'when', 'date posted'],
       'Description': ['description', 'desc', 'details', 'summary', 'info'],
       'Apply Link': ['link', 'url', 'apply', 'apply link', 'application link']
     };
     for (const pattern of restrictionPatterns) {
       const match = taskLower.match(pattern);
       if (match) {
         const mentionedText = match[1];
         const selected = [];
         for (const [colName, aliases] of Object.entries(columnAliases)) {
           if (aliases.some(alias => mentionedText.includes(alias))) {
             selected.push(colName);
           }
         }
         if (selected.length > 0) return selected;
       }
     }
     return defaults;
   }
   ```
