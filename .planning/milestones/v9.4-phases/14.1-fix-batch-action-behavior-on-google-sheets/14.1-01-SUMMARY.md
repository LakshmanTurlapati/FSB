## Decisions Made
- Suppression threshold set to 2+ type actions -- a single type action on Sheets is valid (typing into the active cell)
- URL regex `/docs\.google\.com\/spreadsheets\/d\//i` requires document ID path segment, not just the Sheets homepage
- Used batchHandled flag pattern instead of nested conditionals -- cleaner fall-through to single-action path
- First action from suppressed batch is preserved as aiResponse.actions[0] so no iteration is wasted
- URL check wrapped in try/catch since it's a safety measure, not a blocking requirement
