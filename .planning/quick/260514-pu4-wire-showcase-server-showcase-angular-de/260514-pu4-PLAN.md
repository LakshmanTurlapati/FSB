# Plan: Wire showcase deps into ci.yml extension job

## Context

v0.9.69 milestone shipped on `Refinements` (88 commits ahead of main). The expanded `npm test` chain now includes ~30 tests that require `showcase/server/node_modules` (express-rate-limit, better-sqlite3, ws) or `showcase/angular/node_modules` (chart.js, Angular CLI build chain). CI's `extension` job currently installs only root + mcp deps, so on a fresh runner those tests will fail to require their deps.

This is a one-file change: add two install steps to the `extension` job in `.github/workflows/ci.yml`.

## Task 1: Add showcase deps to ci.yml extension job

**Files:** `.github/workflows/ci.yml`

**Action:**

Locate the `extension` job (lines 14-29 currently). After the "Install mcp deps (needed by root test script)" step (line 22-23), insert two new steps:

```yaml
      - name: Install showcase/server deps (needed by server-telemetry tests in npm test chain)
        run: npm --prefix showcase/server install --no-audit --no-fund
      - name: Install showcase/angular deps (needed by fsb-telemetry-service + build-smoke + privacy-page tests in npm test chain)
        run: npm --prefix showcase/angular install --no-audit --no-fund
```

Keep the existing "Validate extension" + "Run extension + bridge contract tests" steps after.

Do NOT modify the `mcp-smoke` or `website` jobs — they don't need this change.

**Verify:**

1. `grep -c "npm --prefix showcase/server install" .github/workflows/ci.yml` → ≥1
2. `grep -c "npm --prefix showcase/angular install" .github/workflows/ci.yml` → ≥1 (one in extension job + one already in website job = 2 total)
3. `yamllint .github/workflows/ci.yml` (if yamllint is available; otherwise just verify the file still parses by checking step structure with `grep -E "^\s+- name:" .github/workflows/ci.yml | wc -l`)
4. The new steps appear BEFORE "Run extension + bridge contract tests"

**Done when:**

- `.github/workflows/ci.yml` `extension` job has 4 install steps (root, mcp, showcase/server, showcase/angular) before npm test.
- `npm test` on the current local Refinements branch still exits 0 (regression check).
- Commit lands with message `ci: install showcase/server + showcase/angular deps in extension job for v0.9.69 telemetry tests`.
