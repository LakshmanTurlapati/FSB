## Recovery if publish fails

| Failure | Recovery |
|---------|----------|
| Account-age rejection | Wait until the GitHub account passes the 7-day threshold. |
| Name collision (concurrent submission since 2026-05-08) | Edit `skills/FSB Skill/SKILL.md` frontmatter to use the documented fallback `name: fsb-browser` + `displayName: FSB`; rebuild the artifact (`npm run package:skill`); republish. |
| VirusTotal / ClawScan flag | Investigate the specific flag locally; common causes are over-eager pattern matching on benign Node spawn calls in the doctor wrapper. Adjust scripts to remove the flagged pattern (without losing the diagnostic value); rebuild; republish. |
| Auth token expired mid-publish | `clawhub logout && clawhub login` and rerun `clawhub publish "skills/FSB Skill"`. |
