  <action>
Create the directory `.github/workflows/` if it does not exist, then create `.github/workflows/publish-mcp.yml` with the following exact content:

```yaml
name: Publish MCP Server to npm

on:
  push:
    tags:
      - 'v*'

jobs:
  publish:
    runs-on: ubuntu-latest
    permissions:
      contents: read
      id-token: write
    defaults:
      run:
        working-directory: mcp-server
    steps:
      - name: Checkout repository
        uses: actions/checkout@v4

      - name: Setup Node.js
        uses: actions/setup-node@v4
        with:
          node-version: '20'
          registry-url: 'https://registry.npmjs.org'

      - name: Install dependencies
        run: npm ci

      - name: Build TypeScript
        run: npm run build

      - name: Verify shebang
        run: head -1 build/index.js | grep -q '#!/usr/bin/env node' || (echo "ERROR: shebang missing from build/index.js" && exit 1)

      - name: Verify pack contents
        run: |
          npm pack --dry-run 2>&1 | tee /tmp/pack-output.txt
          if grep -q 'src/' /tmp/pack-output.txt; then
            echo "ERROR: src/ files found in pack output"
            exit 1
          fi

      - name: Publish to npm
        run: npm publish --access public
        env:
          NODE_AUTH_TOKEN: ${{ secrets.NPM_TOKEN }}
```

Key design decisions reflected:
- Triggers on `v*` tag push only (per D-01) -- no GitHub Release creation needed (per D-02)
- All npm commands use `working-directory: mcp-server` (per D-03, monorepo structure)
- `npm ci` for reproducible installs (not `npm install`)
- Explicit build step ensures fresh compilation before publish
- Shebang verification catches TypeScript compiler regressions
- Pack contents check prevents accidental source publishing
- `--access public` required for unscoped packages on first publish
- `NPM_TOKEN` secret must be configured manually in GitHub repo settings (Settings > Secrets > Actions > New repository secret)
- Comment at top of file documents the NPM_TOKEN requirement for future maintainers
