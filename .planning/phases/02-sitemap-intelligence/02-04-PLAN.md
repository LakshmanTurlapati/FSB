## Goal

## Tasks

### Task 1: Pre-bundled Site Map Loader

**File:** `background.js`

Add a cached loader for pre-bundled site maps:

```javascript
const bundledSiteMapCache = new Map();

async function loadBundledSiteMap(domain) {
  // Check cache first
  if (bundledSiteMapCache.has(domain)) {
    return bundledSiteMapCache.get(domain);
  }

  // Normalize domain: strip www. prefix for lookup
  const lookupDomain = domain.replace(/^www\./, '');

  // Try exact domain match
  for (const d of [lookupDomain, domain]) {
    try {
      const url = chrome.runtime.getURL(`site-maps/${d}.json`);
      const response = await fetch(url);
      if (response.ok) {
        const data = await response.json();
        bundledSiteMapCache.set(domain, data);
        return data;
      }
    } catch (e) {
      // File not found, continue
    }
  }

  bundledSiteMapCache.set(domain, null);
  return null;
}
```

Add a message handler so AI integration can request it:

```javascript
case 'getSiteMap':
  (async () => {
    const domain = request.domain;

    // Priority 1: Pre-bundled map
    const bundled = await loadBundledSiteMap(domain);
    if (bundled) {
      sendResponse({ success: true, siteMap: bundled, source: 'bundled' });
      return;
    }

    // Priority 2: Memory-stored map (AI refined preferred)
    const memories = await memoryStorage.query({ domain, category: 'site_map' });
    if (memories && memories.length > 0) {
      // Prefer refined over basic
      const refined = memories.find(m => m.typeData?.sitePattern?.refined);
      const best = refined || memories[0];
      sendResponse({ success: true, siteMap: best.typeData.sitePattern, source: refined ? 'memory_refined' : 'memory_basic' });
      return;
    }

    sendResponse({ success: false });
  })();
  return true;
```

**Verify:** `getSiteMap` for a domain with bundled map returns it. For a domain with memory map returns it. For unknown domain returns `{ success: false }`.

### Task 2: Build Site Knowledge Prompt Section

**File:** `ai/ai-integration.js`

New function that formats a site map into a concise prompt section:

```javascript
function formatSiteKnowledge(siteMap, domain) {
  if (!siteMap) return '';

  const lines = [`SITE KNOWLEDGE (${domain}):`];

  // Pages summary
  if (siteMap.pages) {
    const pageEntries = Object.entries(siteMap.pages);
    const pageList = pageEntries.slice(0, 15).map(([path, info]) => {
      const parts = [path];
      if (info.title) parts.push(info.title);
      if (info.formCount > 0) parts.push(`${info.formCount} forms`);
      return parts.join(' - ');
    }).join(', ');
    lines.push(`Pages: ${pageList}`);
  }

  // Navigation
  if (siteMap.navigation && siteMap.navigation.length > 0) {
    const navItems = siteMap.navigation.slice(0, 10).map(n => n.label).join(', ');
    lines.push(`Navigation: ${navItems}`);
  }

  // Workflows (from AI refinement)
  if (siteMap.workflows && siteMap.workflows.length > 0) {
    lines.push('Workflows: ' + siteMap.workflows.slice(0, 5).join('; '));
  }

  // Tips (from AI refinement)
  if (siteMap.tips && siteMap.tips.length > 0) {
    lines.push('Tips: ' + siteMap.tips.slice(0, 5).join('; '));
  }

  // Key selectors (compact)
  if (siteMap.keySelectors) {
    const selectorEntries = Object.entries(siteMap.keySelectors);
    if (selectorEntries.length > 0) {
      const selectorSummary = selectorEntries.slice(0, 5).map(([page, sels]) => {
        const selList = Array.isArray(sels) ? sels.slice(0, 3).join(', ') : sels;
        return `${page}: ${selList}`;
      }).join('; ');
      lines.push(`Key selectors: ${selectorSummary}`);
    }
  }

  return lines.join('\n');
}
```

**Budget:** The site knowledge section should fit within ~500-800 characters, carved from the memory portion of the 15K budget (10% = 1500 chars, site knowledge gets up to half of that).

**Verify:** `formatSiteKnowledge(mockSiteMap, 'linkedin.com')` returns a compact multi-line string under 800 chars.

### Task 3: Inject Site Knowledge into buildPrompt()

**File:** `ai/ai-integration.js`

In `buildPrompt()` (or `buildMemoryContext()`), add site knowledge retrieval at the start of a task:

```javascript
// In buildPrompt or buildMemoryContext, near where memory context is assembled:

// Retrieve site map for current domain
let siteKnowledge = '';
if (sessionState?.currentUrl) {
  try {
    const domain = new URL(sessionState.currentUrl).hostname;
    const mapResult = await chrome.runtime.sendMessage({
      action: 'getSiteMap',
      domain
    });
    if (mapResult && mapResult.success) {
      siteKnowledge = formatSiteKnowledge(mapResult.siteMap, domain);
    }
  } catch (e) {
    // Non-critical, proceed without site knowledge
  }
}

// Include in the memory section of the prompt
if (siteKnowledge) {
  memorySection = siteKnowledge + '\n\n' + memorySection;
}
```

**Important:** Only inject on the FIRST iteration of a session (or when domain changes). Don't re-inject on every iteration -- it wastes prompt budget.

**Verify:** Start a task on a domain with a site map -> check the assembled prompt in debug logs -> "SITE KNOWLEDGE" section present. Start on unknown domain -> no section.
