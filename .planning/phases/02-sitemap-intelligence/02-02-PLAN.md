## Tasks

### Task 1: Create AI Refinement Module

**File:** `lib/memory/sitemap-refiner.js` (NEW)

Module that takes a Tier 1 site map + raw research data and sends a single focused prompt to the user's configured AI model.

**Function:** `async function refineSiteMapWithAI(sitePattern, researchData, config)`

**Prompt design:**
- System: "You are a web automation expert. Analyze this site map and provide navigation intelligence."
- User content: Tier 1 sitePattern (pages, navigation, forms) + page titles + form descriptions (NOT raw element arrays)
- Requested output (JSON):
  ```json
  {
    "workflows": [
      "Job search: /jobs -> use search form -> click result -> /jobs/view/{id}"
    ],
    "tips": [
      "Uses AJAX loading on /jobs, wait for elements after filter changes",
      "Login requires email first, then password on separate screen"
    ],
    "pagePurposes": {
      "/": "Landing page with featured content and global navigation",
      "/jobs": "Job search with filters and result listing"
    },
    "selectorPreferences": {
      "note": "Site uses data-testid consistently, prefer those selectors"
    },
    "navigationStrategy": "Top navbar is persistent, use it for primary navigation"
  }
  ```

**AI call mechanics:**
- Use the same provider/model from user's config (`chrome.storage.local` settings)
- Use the existing `UniversalProvider` class pattern from `ai/ai-integration.js`
- Single call, not per-page
- Timeout: 30 seconds
- Token estimate: ~2-3K input, ~500-1K output

**Merge logic:**
- On success: merge AI output into sitePattern, set `refined: true`, `refinedAt: Date.now()`
- On failure: return original sitePattern unchanged (Tier 1 result preserved)
- Update the memory in storage via `memoryStorage.update()`

**Verify:** Call `refineSiteMapWithAI()` with a valid sitePattern -> returns enriched pattern with workflows and tips populated, `refined: true`.

### Task 3: Auto-trigger Tier 2 After Tier 1

**File:** `ui/options.js`

Modify `saveResearchToMemory()` (created in Plan 2-01, Task 4):

```javascript
async function saveResearchToMemory(researchId) {
  // ... existing Tier 1 logic ...
  const sitePattern = convertToSiteMap(research);
  const memory = createSiteMapMemory(domain, sitePattern);
  await memoryStorage.store(memory);

  // Tier 2: AI refinement if toggle is ON
  const settings = await chrome.storage.local.get(['autoRefineSiteMaps']);
  if (settings.autoRefineSiteMaps !== false) {
    showToast('Refining site map with AI...', 'info');
    try {
      const refined = await refineSiteMapWithAI(sitePattern, research);
      // Update memory with refined data
      memory.typeData.sitePattern = refined;
      memory.text = `Site map for ${domain}: ${refined.pageCount} pages, ${refined.formCount} forms (AI enhanced)`;
      await memoryStorage.update(memory.id, memory);
      showToast('Site map saved and refined for ' + domain, 'success');
    } catch (err) {
      console.warn('AI refinement failed, keeping Tier 1 result:', err.message);
      showToast('Site map saved for ' + domain + ' (AI refinement skipped)', 'info');
    }
  } else {
    showToast('Site map saved to memory for ' + domain, 'success');
  }
}
```

**Verify:** With toggle ON: save to memory -> brief "Refining..." toast -> "saved and refined" toast -> memory has `refined: true`. With toggle OFF: save -> only Tier 1 -> `refined: false`.
