**Function 1: formatToolsForProvider(tools, provider)**

Takes array of tool definition objects (from TOOL_REGISTRY), returns provider-formatted tools.

```javascript
// OpenAI/xAI/OpenRouter/Custom (default):
tools.map(t => ({
  type: 'function',
  function: {
    name: t.name,
    description: t.description,
    parameters: t.inputSchema
  }
}))

// Anthropic:
tools.map(t => ({
  name: t.name,
  description: t.description,
  input_schema: t.inputSchema
}))

// Gemini:
[{
  functionDeclarations: tools.map(t => ({
    name: t.name,
    description: t.description,
    parameters: t.inputSchema
  }))
}]
```

**Function 2: parseToolCalls(response, provider)**

Extracts normalized tool calls from a raw API response. Returns array of `{ id, name, args }` objects where `args` is always a parsed object (never a string).

```javascript
// OpenAI/xAI/OpenRouter/Custom (default):
// Location: response.choices[0].message.tool_calls
// ID: tool_calls[i].id
// Name: tool_calls[i].function.name
// Args: JSON.parse(tool_calls[i].function.arguments)  <-- STRING, must parse
// PITFALL: arguments is a JSON STRING, must JSON.parse it
const calls = response.choices?.[0]?.message?.tool_calls || [];
return calls.map(c => ({
  id: c.id,
  name: c.function.name,
  args: typeof c.function.arguments === 'string'
    ? JSON.parse(c.function.arguments)
    : c.function.arguments
}));

// Anthropic:
// Location: response.content[] where type === 'tool_use'
// ID: block.id
// Name: block.name
// Args: block.input  <-- ALREADY PARSED OBJECT, do NOT JSON.parse
const blocks = (response.content || []).filter(b => b.type === 'tool_use');
return blocks.map(b => ({
  id: b.id,
  name: b.name,
  args: b.input   // Already an object per Pitfall 1
}));

// Gemini:
// Location: response.candidates[0].content.parts[] where functionCall exists
// ID: part.functionCall.id || generate a fallback ID (Gemini may not provide id)
// Name: part.functionCall.name
// Args: part.functionCall.args  <-- ALREADY PARSED OBJECT
const parts = response.candidates?.[0]?.content?.parts || [];
return parts.filter(p => p.functionCall).map((p, i) => ({
  id: p.functionCall.id || `gemini-call-${i}`,
  name: p.functionCall.name,
  args: p.functionCall.args || {}  // Already an object per Pitfall 1
}));
```

**Function 3: formatToolResult(id, result, provider, options)**

Formats a tool execution result for appending to conversation history.
- `id`: string tool call ID from parseToolCalls
- `result`: object with the execution result
- `provider`: provider key string
- `options`: optional { name: string, isError: boolean }

```javascript
// OpenAI/xAI/OpenRouter/Custom (default):
// Role: "tool"
// Content: JSON string of result
{
  role: 'tool',
  tool_call_id: id,
  content: typeof result === 'string' ? result : JSON.stringify(result)
}

// Anthropic:
// Role: "user"
// Content: array with tool_result block
// PITFALL 3: Must use role: "user", NOT role: "tool"
{
  role: 'user',
  content: [{
    type: 'tool_result',
    tool_use_id: id,
    content: typeof result === 'string' ? result : JSON.stringify(result),
    ...(options?.isError ? { is_error: true } : {})
  }]
}

// Gemini:
// Role: "user"
// Content: functionResponse part
// PITFALL 3: Must use role: "user", NOT role: "tool"
{
  role: 'user',
  parts: [{
    functionResponse: {
      name: options?.name || '',
      response: typeof result === 'object' ? result : { result }
    }
  }]
}
```

**Function 4: isToolCallResponse(response, provider)**

Returns boolean indicating if the response contains tool calls.

```javascript
// OpenAI/xAI/OpenRouter/Custom (default):
// Signal: finish_reason === 'tool_calls'
const fr = response.choices?.[0]?.finish_reason;
return fr === 'tool_calls';

// Anthropic:
// Signal: stop_reason === 'tool_use'
return response.stop_reason === 'tool_use';

// Gemini:
// PITFALL 2: finishReason does NOT signal tool calls
// Must inspect parts for functionCall presence
const parts = response.candidates?.[0]?.content?.parts || [];
return parts.some(p => !!p.functionCall);
```

**Function 5: formatAssistantMessage(response, provider)**

Extracts the assistant's message (including tool call blocks) for conversation history. This must be appended to history BEFORE tool results per Pitfall 5.

```javascript
// OpenAI/xAI/OpenRouter/Custom (default):
// The full message object from choices[0].message
const msg = response.choices?.[0]?.message;
return { role: 'assistant', ...msg };

// Anthropic:
// The full content array (contains both text and tool_use blocks)
return {
  role: 'assistant',
  content: response.content || []
};

// Gemini:
// The full content object with parts
return {
  role: 'model',  // Gemini uses 'model' not 'assistant'
  parts: response.candidates?.[0]?.content?.parts || []
};
```
