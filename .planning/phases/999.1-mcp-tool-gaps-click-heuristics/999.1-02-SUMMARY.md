## Accomplishments

## Decisions Made
- Used TreeWalker instead of querySelectorAll('*') for efficient DOM traversal -- TreeWalker's FILTER_REJECT prunes entire invisible subtrees, avoiding unnecessary computation on hidden DOM branches
- Preferred deepest visible match (iterating candidates in reverse) -- the most nested element containing the search text is closest to the actual text content, yielding more precise clicks
- Made required: [] instead of requiring one of selector/text -- simpler schema, and the handler naturally falls through to coordinate fallback if neither finds an element
