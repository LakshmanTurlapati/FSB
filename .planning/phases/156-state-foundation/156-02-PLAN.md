class TranscriptStore {
  /**
   * @param {Object} options
   * @param {number} options.tokenBudget - Max token budget (default 128000)
   * @param {number} options.compactThreshold - Fraction of budget that triggers compaction (default 0.8)
   * @param {number} options.keepRecentCount - Number of recent tool_result messages to keep intact (default 5)
   */
  constructor(options = {}) {
    this.messages = [];
    this.tokenBudget = options.tokenBudget || 128000;
    this.compactThreshold = options.compactThreshold || 0.8;
    this.keepRecentCount = options.keepRecentCount || 5;
    this._compactionCount = 0;
    this._totalTokensSaved = 0;
  }
}
```

3. **`append(message)`** method -- pushes a message object onto this.messages. Returns the store instance for chaining.

4. **`compact()`** method -- extract the EXACT logic from `compactHistory()` in agent-loop.js lines 262-342. This is the core extraction. The method must:
   - Check if `estimateTokens(this.messages)` exceeds `this.tokenBudget * this.compactThreshold`

   - Replace each compactable tool_result with a one-liner summary: `"${toolName} returned ${status}"`
   - Increment `this._compactionCount` and track tokens saved in `this._totalTokensSaved`
   - Return `{ compacted: true, removedCount, estimatedTokens: newTokens }`
   Copy the tool name extraction and status detection logic EXACTLY from compactHistory (lines 293-334).

function createTurnResult(fields) {
  return {
    sessionId: fields.sessionId || '',
    iteration: fields.iteration || 0,
    inputTokens: fields.inputTokens || 0,
    outputTokens: fields.outputTokens || 0,
    cost: fields.cost || 0,
    matchedTools: fields.matchedTools || [],       // Array<string> -- tool names called this turn
    toolResults: fields.toolResults || [],          // Array<{name, success, hadEffect}>

    errorMessage: fields.errorMessage || null,     // Error text if stopReason is ERROR

    durationMs: fields.durationMs || 0             // Time for this iteration's API call
  };
}
```

1. **`createActionEvent(fields = {})`** factory function:
Returns a structured action event:
```javascript
function createActionEvent(fields) {
  return {
    tool: fields.tool || 'unknown',
    params: fields.params || {},
    result: {
      success: fields.result?.success ?? false,
      hadEffect: fields.result?.hadEffect ?? false,
      error: fields.result?.error || null
    },
    timestamp: fields.timestamp || Date.now(),
    iteration: fields.iteration || 0,
    durationMs: fields.durationMs || 0
  };
}
```

This matches the shape currently pushed inline at agent-loop.js line 1291-1297, but adds `durationMs` and normalizes the `result.error` field.

2. **ActionHistory class**:
```javascript
class ActionHistory {
  constructor() {
    this.events = [];
  }
}
```

3. **`push(eventOrFields)`** method:
   - If argument is already a structured event (has `tool` and `timestamp`), push it directly
   - Otherwise, pass through `createActionEvent(eventOrFields)` to normalize
   - Returns the created event

4. **`getLastN(n)`** method:
   - Returns the last N events (or all if fewer than N exist)
   - Returns a new array (not a reference)

5. **`getByIteration(iteration)`** method:
   - Returns all events for a specific iteration number
   - Used for replay and per-turn analysis

6. **`getToolCounts()`** method:
   - Returns an object mapping tool names to call counts: `{ click: 5, type: 3, navigate: 1 }`
   - Used by stuck detection to identify repetitive tool patterns

7. **`getFailures()`** method:
   - Returns all events where result.success === false
   - Used for error analysis and recovery

8. **`diff(fromIndex)`** method:
   - Returns events added since a given index: `this.events.slice(fromIndex)`
   - Enables delta comparison between turns (per STATE-04 requirement)

9. **`hydrate(events)`** method:
   - Replaces this.events with provided array, normalizing each through createActionEvent
   - Used when restoring from storage

10. **`toJSON()`** method:
    - Returns a JSON-serializable copy of this.events
    - Handles any non-serializable values in params (truncates large objects)

11. **`clear()`** method:
    - Empties events array, returns old events

12. **Stats getter `get length()`**:
    - Returns `this.events.length`

Use `var` for module-level variables. Standard cross-environment export:

if (typeof module !== 'undefined' && module.exports) {

}
