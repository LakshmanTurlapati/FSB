Add a sanitizeOverlayText helper function INSIDE the sessionStatus case block (or just above it, still within the message listener scope). This function:
1. Strips markdown formatting: remove `**`, `*`, `__`, `_`, `` ` ``, `#` prefixes, `[text](url)` link syntax
2. Clamps text length to 80 characters with ellipsis if longer
3. Returns the cleaned string

```js
function sanitizeOverlayText(text) {
  if (!text || typeof text !== 'string') return text;
  let clean = text
    .replace(/\*\*(.+?)\*\*/g, '$1')   // **bold**
    .replace(/\*(.+?)\*/g, '$1')        // *italic*
    .replace(/__(.+?)__/g, '$1')        // __bold__
    .replace(/_(.+?)_/g, '$1')          // _italic_
    .replace(/`(.+?)`/g, '$1')          // `code`
    .replace(/^#{1,6}\s+/gm, '')        // # headings
    .replace(/\[([^\]]+)\]\([^)]+\)/g, '$1')  // [text](url)
    .trim();
  if (clean.length > 80) clean = clean.substring(0, 77) + '...';
  return clean;
}
```
