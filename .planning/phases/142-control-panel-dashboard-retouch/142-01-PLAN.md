# Phase 142 Plan 01: Control Panel Surface Retouch

## Objective

Retouch the control panel shell, navigation, cards, forms, and settings surfaces so they align with the shared UI baseline established in Phase 140 and the operator-facing polish established in Phase 141.

## Tasks

1. Normalize dark-mode shell and navigation chrome across the control panel header, sidebar, and content surfaces.
2. Tighten card, form, and utility-section spacing so the dashboard reads cleaner and denser without redesigning the product.
3. Clean up user-facing labels and chrome inconsistencies in the control panel navigation and settings/help surfaces.

## Acceptance Criteria

- Control panel header, sidebar, nav items, forms, and cards use one coherent visual system.
- Dark mode no longer carries the earlier warm/orange shell tint.
- Navigation and settings surfaces feel flatter, cleaner, and more consistent with the sidepanel/popup direction.
