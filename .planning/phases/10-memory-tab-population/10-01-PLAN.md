3. **Create procedural memory if session completed AND actionHistory has 2+ successful actions:**
   - Text: `"Workflow for '<task>': N steps"` (truncate task to 60 chars)
   - typeData: steps array of `"tool(param)"` strings from last 10 successful actions, selectors from workingSelectors values
   - confidence: 0.7, tags: ['workflow', 'auto-extracted']
   - Use actionHistory entries where `a.result?.success !== false` (accounts for undefined success field)

Use `createEpisodicMemory`, `createSemanticMemory`, `createProceduralMemory` from memory-schemas.js (they are globally available via self).

let enrichedSection = '';
if (session._enrichedData) {

  if (ed.hardFacts?.workingSelectors) {
    const ws = Object.entries(ed.hardFacts.workingSelectors);
    if (ws.length > 0) {
