2. **Agent list rendering -- `loadAgentListUI()`:**
   ```javascript
   async function loadAgentListUI() {
     const listEl = document.getElementById('agentListSP');
     const emptyEl = document.getElementById('agentEmptyState');
     if (!listEl) return;

     listEl.innerHTML = '<div style="text-align:center;padding:20px;color:var(--text-secondary)"><i class="fa fa-spinner fa-spin"></i> Loading...</div>';
     emptyEl.style.display = 'none';

     try {
       const response = await new Promise(resolve => {
         chrome.runtime.sendMessage({ action: 'listAgents' }, resolve);
       });
       const agents = response?.agents || [];

       if (agents.length === 0) {
         listEl.innerHTML = '';
         emptyEl.style.display = 'flex';
         return;
       }

       emptyEl.style.display = 'none';
       listEl.innerHTML = agents.map(agent => renderAgentRow(agent)).join('');

       // Attach event listeners to action buttons
       listEl.querySelectorAll('[data-action]').forEach(btn => {
         btn.addEventListener('click', handleAgentAction);
       });
     } catch (error) {
       listEl.innerHTML = '<div style="padding:20px;color:#c62828;">Failed to load agents</div>';
     }
   }
   ```

3. **Agent row renderer -- `renderAgentRow(agent)`:**
   Returns HTML string for one agent row:
   - Header: agent.name + status badge (determine status: if agent has a running flag or lastRunStatus === 'running' use 'running', else agent.enabled ? 'active' : 'paused')
   - Info line: schedule description (use formatScheduleShort(agent.schedule)), last run time (agent.lastRunAt ? relative time or toLocaleString : 'Never'), run count
   - Actions row: three buttons with data-action and data-agent-id attributes:
     - Run Now: `<button class="agent-action-btn" data-action="run" data-agent-id="${agent.agentId}"><i class="fa fa-play"></i> Run</button>`
     - Pause/Resume: `<button class="agent-action-btn" data-action="toggle" data-agent-id="${agent.agentId}"><i class="fa fa-${agent.enabled ? 'pause' : 'play-circle'}"></i> ${agent.enabled ? 'Pause' : 'Resume'}</button>`
     - Delete: `<button class="agent-action-btn danger" data-action="delete" data-agent-id="${agent.agentId}"><i class="fa fa-trash"></i></button>`
   - Wrap in a div with class="agent-row" and data-agent-id attribute
   - Add a hidden expandable section div.agent-row-detail (empty for now, Plan 02 fills it)

   - For 'delete': show confirm() dialog "Delete this agent?", if confirmed sendMessage({action: 'deleteAgent', agentId}), on response reload list
