       - For site_pattern category: show pattern summary from `memory.typeData.sitePattern`

       - Validated At: formatted date from `memory.typeData.validatedAt` (if present)

       - Steps: numbered list from `memory.typeData.steps`
       - Selectors Used: code-formatted list from `memory.typeData.selectors`
       - Success Rate: `memory.typeData.successRate` as percentage with color (green >80%, orange 50-80%, red <50%)
       - Total Runs: `memory.typeData.totalRuns`
       - Target URL: `memory.typeData.targetUrl` (if present)

       - Wrapped in a section with header "AI Analysis" and a subtle indicator icon

       - Update the click handler section to call `toggleMemoryDetail(item)` for all items (which internally delegates to `toggleMemoryGraph` for site_maps)
       - Add `cursor: pointer` style to all memory items

       - The chevron should rotate 90 degrees when expanded (CSS handles this)
