Create site-guides/ecommerce/mega-menu.js (ecommerce directory already exists).

The site guide targets CSS hover-triggered mega-menu navigation on e-commerce and large content sites. Primary test targets: Best Buy (bestbuy.com has robust mega-menus under "Products" and "Brands" nav items), Home Depot (homedepot.com), and Lowe's (lowes.com). Secondary: any site with a horizontal nav bar where hovering reveals a multi-column dropdown panel.

MEGA-MENU INTERACTION CHALLENGE:
Mega-menus open when the user hovers over a top-level nav item. They close when the mouse leaves the nav item area OR the mega-menu panel area. The critical challenge is maintaining hover state while moving from the nav item into the mega-menu panel and then to a specific nested sub-link. Two factors:

1. CSS :hover pseudo-class: Triggered by the browser's actual mouse cursor position, NOT by JavaScript events. The MCP `hover` tool dispatches JS mouseenter/mouseover events which DO NOT move the browser's actual cursor. However, most modern mega-menus use JavaScript event listeners (mouseenter/mouseleave) rather than pure CSS :hover, so the JS hover tool works on MOST sites.

2. Hover path: Moving the mouse diagonally from the nav item to a sub-link may cross other nav items, triggering THEIR mega-menus instead. Some mega-menus implement a "hover intent" delay (200-400ms) to mitigate this.

TWO INTERACTION STRATEGIES:

STRATEGY A (DOM hover + click -- preferred, works on JS-based menus):
1. Use `hover` tool on the top-level nav item CSS selector
2. Wait 300-500ms for the mega-menu animation to complete
3. Use `click` tool on the nested sub-link CSS selector (menu stays open during same-document DOM interactions)

STRATEGY B (CDP coordinate path -- for pure CSS :hover menus where Strategy A fails):
1. Use `get_dom_snapshot` to find coordinates of the top-level nav item
2. Use `drag` with startX,startY = nav item center and endX,endY = 1px below (zero-distance move). This sends CDP mouseMoved events that trigger CSS :hover.
3. Wait 300-500ms for animation
4. Use `click_at` at the coordinates of the nested sub-link inside the mega-menu

TIMING CONSIDERATIONS:
