## Background

CI run 25780666786 surfaced a pre-existing failure in the `extension` job (also failed on prior `feat/showcase-i18n` push 25750958537). Two assertions in `tests/remote-control-rebrand.test.js` fail:

- `[SYNC-05] about-page architecture names Sync as the paired extension control surface`
- `[SYNC-05] about-page arch-box renamed to "Sync"`

Test regexes:
```js
/<h4>Sync<\/h4>\s*<p>Live extension control surface, paired via QR<\/p>/
/<h4>Sync<\/h4>/
```

Current about-page markup (Phase 262):
```html
<h4 i18n="@@about.arch.box.sync.title">Sync</h4>
<p i18n="@@about.arch.box.sync.desc">Live extension control surface, paired via QR</p>
```

Phase 262 added `i18n="..."` attributes to the `<h4>` and `<p>` tags as required by COPY-01. The literal `<h4>Sync</h4>` regex no longer matches because the tag now carries the i18n attribute. Test intent is unchanged: assert the about-page still names "Sync" as the paired extension control surface.
