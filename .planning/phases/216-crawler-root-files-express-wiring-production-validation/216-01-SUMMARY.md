## Deviations from Plan

None. Plan executed exactly as written. All four tasks landed in declared files; verify-static.sh and the umbrella verify.sh both exit 0 against Plan 01 outputs.

One minor portability adjustment was made inside the verify script (not a deviation from the plan, since the plan only specified intent, not the regex flag): the ASCII-only check uses `LC_ALL=C grep '[^[:print:][:space:]]'` instead of `grep -P "[^\x00-\x7F]"`, because BSD grep on macOS does not support `-P`.
