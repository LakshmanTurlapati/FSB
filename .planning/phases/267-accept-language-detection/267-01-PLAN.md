## Tasks (sequential)

### Task 1 -- Middleware module
**File:** `showcase/server/src/middleware/accept-language.js`
**Exports:**
- `pickBestLocale(headerValue, supported)` -- pure parser; returns best supported locale code or `null`.
- `createAcceptLanguageMiddleware({ supported, defaultLocale, cookieName })` -- factory returning Express middleware.

**Parser algorithm:**
1. If header is null/empty -> return null.
2. Split on `,`, for each entry: trim, split first `;`, parse `q=` (default 1.0). Drop malformed.
3. Sort by quality desc, then original order (stable).
4. For each entry's tag (lowercased): apply alias map -> if it matches a `supported` locale, return it; else try primary subtag (`es-MX` -> `es`).
5. Return null if nothing matches.

**Alias map:**
```
zh-hans*, zh-cn, zh-sg, zh -> zh-CN
zh-hant*, zh-tw, zh-mo, zh-hk -> zh-TW
```

**Middleware behavior:**
1. Only act on `GET`/`HEAD` of `/`.
2. Parse `Cookie` header inline; if `fsb-locale=<valid>` (in supported), `next()`.
3. Call `pickBestLocale(req.headers['accept-language'], supported)`. If null or === defaultLocale, `next()`.
4. Else `res.redirect(302, '/' + best + '/')`.
5. Wrap parsing in try/catch -- any error -> `next()` (T-267-04 mitigation).

### Task 3 -- Wire middleware into server.js
**File:** `showcase/server/server.js`
**Action:** Insert middleware BEFORE the `marketingRoutes` block (around line 172). Configure with:
```js
const { createAcceptLanguageMiddleware } = require('./src/middleware/accept-language');
const SUPPORTED_LOCALES = ['es', 'de', 'ja', 'zh-CN', 'zh-TW'];
app.use(createAcceptLanguageMiddleware({
  supported: SUPPORTED_LOCALES,
  defaultLocale: 'en',
  cookieName: 'fsb-locale',
}));
```
