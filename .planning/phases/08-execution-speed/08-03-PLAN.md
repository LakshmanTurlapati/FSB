Add DETERMINISTIC_PATTERNS constant and detection functions near calculateActionDelay.

DETERMINISTIC_PATTERNS array:
```javascript
const DETERMINISTIC_PATTERNS = [
  {
    name: 'formFill',
    detect: (actions) => {
      // All type actions to different selectors
      return actions.every(a => a.tool === 'type') &&
             new Set(actions.map(a => a.params?.selector)).size === actions.length;
    },
    optimize: true,
    minDelay: 50  // Between batched actions
  },
  {
    name: 'clickType',
    detect: (actions) => {
      // Click followed by type (click input, then type)
      return actions.length === 2 &&
             actions[0].tool === 'click' &&
             actions[1].tool === 'type';
    },
    optimize: true,
    minDelay: 100
  },
  {
    name: 'multiClick',
    detect: (actions) => {
      // Multiple clicks to different elements (e.g., checkbox selections)
      return actions.every(a => a.tool === 'click') &&
             new Set(actions.map(a => a.params?.selector)).size === actions.length &&
             actions.length <= 3;
    },
    optimize: true,
    minDelay: 100
  }
];
```

Add detectDeterministicPattern function:
1. Takes actions array
2. Returns first matching pattern from DETERMINISTIC_PATTERNS or null

Add executeDeterministicBatch async function:
1. Takes (actions, session, tabId)
2. If no pattern matched, return null (caller should execute normally)
3. Execute each action with pattern.minDelay between them
4. Collect results
5. Return { batched: true, pattern: pattern.name, results, count: actions.length }

Changes to automation loop:

1. At start of iteration, check for pending prefetch:
```javascript
let domResponse;
if (pendingDOMPrefetch) {
  try {
    domResponse = await pendingDOMPrefetch;
    pendingDOMPrefetch = null;
  } catch (e) {
    // Prefetch failed, fetch normally
    domResponse = await fetchDOM(session.tabId);
  }
} else {
  domResponse = await fetchDOM(session.tabId);
}
```

2. Start AI call and prefetch in parallel:
```javascript
const aiPromise = callAIAPI(...);
pendingDOMPrefetch = prefetchDOM(session.tabId);
const aiResponse = await aiPromise;
```

3. Before executing actions, check for deterministic batch:
```javascript
const actions = aiResponse.actions || [];
if (actions.length > 1) {
  const batchResult = await executeDeterministicBatch(actions, session, session.tabId);
  if (batchResult) {
    // Batch executed successfully
    // Log and continue to next iteration
    continue;
  }
}
// Fall through to normal action execution
```

Key decision: Prefetch starts AFTER AI call begins (not before) so DOM reflects any changes from current iteration's actions.
