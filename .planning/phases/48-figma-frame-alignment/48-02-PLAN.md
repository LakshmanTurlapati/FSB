**Step 1: Navigate to Excalidraw**
Use MCP `navigate` to open `https://excalidraw.com/`. Wait for the canvas to load using `wait_for_stable`.

**Step 2: Inspect DOM for toolbar and canvas structure**
Use `get_dom_snapshot` to discover:
- The main canvas element (likely a `<canvas>` element)
- The toolbar/toolbox area with shape tool buttons
- Keyboard shortcut availability (Excalidraw uses single-key shortcuts: R=rectangle, F=frame)
- Any welcome dialogs or modals that need dismissal
- Record all discovered CSS selectors

**Step 3: Create a frame (if Excalidraw supports frames)**
- Try `press_key` with key `f` to activate the frame tool
- If frame tool exists, draw a frame on the canvas using `drag` (e.g., drag from (300, 200) to (700, 500) to create a ~400x300 frame)
- If Excalidraw does not have a frame tool, use a large rectangle as the "frame" instead (press R, then drag)
- Verify by checking DOM for new shape elements

**Step 4: Draw 3 rectangles inside the frame**
- Press `r` via `press_key` to activate rectangle tool
- Draw rectangle 1 using `drag` from approximately (320, 220) to (420, 320) -- upper-left area of frame
- Press `r` again to re-activate rectangle tool (Excalidraw may deselect after drawing)
- Draw rectangle 2 using `drag` from approximately (350, 280) to (450, 380) -- middle area, deliberately misaligned
- Press `r` again
- Draw rectangle 3 using `drag` from approximately (380, 340) to (480, 440) -- lower area, deliberately misaligned
- After each draw, use `get_dom_snapshot` to verify new elements appeared

**Step 5: Multi-select all 3 shapes**
Try these approaches in order:
1. **Ctrl+A** via `press_key` with key `a` and ctrl=true -- selects all shapes (simplest, but also selects the frame)
2. **Shift+click** via `click_at` with shift=true on each of the 3 rectangles (if their positions are known from Step 4 DOM snapshots)
3. **Drag-select box** via `drag` over the area containing all 3 rectangles (if selection tool is active via pressing `v`)

Verify multi-selection by checking DOM for selection indicators.

**Step 6: Align shapes using toolbar buttons**
- After multi-selecting, look for alignment toolbar/buttons in the DOM using `get_dom_snapshot`
- Excalidraw shows alignment options in the top toolbar or a floating menu when multiple shapes are selected
- Look for buttons with text/aria-label containing "align left", "align center", "align top", "distribute", or similar
- Click the alignment button(s) using `click` (DOM click) since toolbar buttons are HTML elements
- Try at least one alignment action (e.g., align-left or center-horizontally)
- Verify the alignment took effect (DOM positions changed or visual confirmation)

**Step 7: Create the site guide**
