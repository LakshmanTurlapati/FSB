<objective>
Add modifier key support (shift, ctrl, alt) to click_at and drag MCP tools so Excalidraw multi-select (shift+click) and constrained drawing work via CDP trusted events.

Purpose: Excalidraw requires shift+click for multi-selecting shapes before alignment. The current click_at and drag tools dispatch CDP mouse events without modifier keys, making multi-select impossible. Chrome's CDP Input.dispatchMouseEvent supports a modifiers bitmask (1=Alt, 2=Ctrl, 4=Meta, 8=Shift). Adding this plumbing enables shift+click for Phase 48 and benefits all future phases needing modified mouse interactions.

Output: Updated click_at and drag MCP tools accepting optional shift/ctrl/alt boolean params, CDP handlers computing and passing the modifiers bitmask.
</objective>
