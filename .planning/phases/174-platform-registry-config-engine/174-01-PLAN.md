**3. `PLATFORMS` exported const object:**

**4. `resolvePlatformConfig(platformKey)` exported function:**
Takes a platform key string (e.g., `'cursor'`), returns the resolved config path for the current OS, or `null` if the platform is `cli` format or the current OS is not in the configPath map.

```javascript
export function resolvePlatformConfig(platformKey) {
  const platform = PLATFORMS[platformKey];
  if (!platform || !platform.configPath) return null;
  return platform.configPath[process.platform] || null;
}
```

**5. `getPlatformFlags()` exported function:**
Returns an array of all platform flag names (the keys of PLATFORMS). Useful for CLI help text.

```javascript
export function getPlatformFlags() {
  return Object.keys(PLATFORMS);
}
```
